# 00 · 基本使用 ⭐ 新手第一課

學會日常使用 Claude Code 的核心操作，再進入後面的設定課程。

**預估時間：** 30 分鐘  
**前置需求：** 已安裝 Claude Code（`npm install -g @anthropic-ai/claude-code`）

---

## 你會做到什麼

- 下出讓 Claude 真正有用的指令（給上下文、定義完成條件）
- 用 `@` 引用檔案、`ESC` 中斷、`!` 執行 shell 指令
- 養成上下文管理習慣：`/clear`、`/compact`、`claude --continue`
- 跟 git 配合安全工作：先看 `git diff`，改壞了能還原

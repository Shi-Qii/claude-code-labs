# 01 · CLAUDE.md

讓 Claude 自動認識你的專案，不用每次開新對話都重新介紹一遍。

**預估時間：** 15 分鐘  
**前置需求：** 無

---

## 你會做到什麼

- 寫一份 CLAUDE.md，讓 Claude 自動載入你的專案背景、技術棧、開發規範
- 驗證 Claude 能從 CLAUDE.md 回答專案問題，不需要你在對話裡重新說明
- 設定全域 `~/.claude/CLAUDE.md`，讓個人偏好套用到所有專案

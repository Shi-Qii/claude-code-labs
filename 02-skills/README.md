# 02 · 自訂 Slash Commands & Skills

建立你自己的 `/指令`，一行搞定重複的工作流程。

**預估時間：** 20 分鐘  
**前置需求：** 無

---

## 你會做到什麼

- 建立自訂 slash commands（如 `/pr`、`/check`），讓 Claude 執行你定義的多步驟任務
- 讓指令接受參數（`/explain src/UserService.java`）
- 了解 slash commands 與 Skills（SKILL.md）的差異：前者手動觸發、後者 Claude 自動取用

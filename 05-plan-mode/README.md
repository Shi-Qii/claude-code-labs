# 05 · Plan Mode + Sub-agents

複雜任務先讓 Claude 給計畫再動手，多個搜尋任務平行執行。

**預估時間：** 30 分鐘  
**前置需求：** 無

---

## 你會做到什麼

- 用 `Shift+Tab` 切換進入 Plan Mode，或啟動時加 `--permission-mode plan`
- 讓 Claude 先列出影響範圍和風險，確認後再執行
- 用 sub-agents 同時平行搜尋多個不相關的問題，觀察整合後的結果

# 06 · MCP 工具

讓 Claude 直接查 GitHub、抓網頁、讀指定目錄，不用你在中間複製貼上。

**預估時間：** 30 分鐘  
**前置需求：** Node.js、`brew install uv`

---

## 你會做到什麼

- 用 `claude mcp add` 安裝 fetch MCP，讓 Claude 直接抓取任何網頁或 API
- 在 Claude Code 輸入 `/mcp` 確認 MCP Server 連線狀態
- 安裝 GitHub 官方 MCP，讓 Claude 直接查你的 repo issue 和 PR

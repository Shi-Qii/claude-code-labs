# 07 · Schedule

設定雲端排程，讓 Claude 每天自動產生工作摘要、每週掃描 TODO，不用手動觸發。

**預估時間：** 20 分鐘  
**前置需求：** 無（若要用 GitHub MCP 範例，建議先完成 06）

---

## 你會做到什麼

- 用 `/schedule` 建立一個定時任務，讓 Claude 在指定時間自動執行
- 用 `/schedule list` 確認排程存在，並手動觸發一次測試
- 了解 Schedule（雲端執行，無法存取本機檔案）與 Hooks（本機執行）的適用場景差異

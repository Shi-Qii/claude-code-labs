# Claude Code Labs

給同事自學 Claude Code 的實戰練習集。每個單元獨立，照順序做完就能掌握 Claude Code 的核心功能。

## 練習地圖

| # | 主題 | 學到什麼 |
|---|------|---------|
| [01](./01-claude-md/guide.md) | CLAUDE.md | 讓 Claude 認識你的專案 |
| [02](./02-skills/guide.md) | Skills（自訂指令） | 建立你自己的 slash 指令 |
| [03](./03-memory/guide.md) | Memory | 讓 Claude 跨對話記住事情 |
| [04](./04-hooks/guide.md) | Hooks | 自動化：工具執行前後觸發動作 |
| [05](./05-plan-mode/guide.md) | Plan Mode + Sub-agents | 規劃複雜任務、拆分執行 |
| [06](./06-mcp/guide.md) | MCP 工具 | 串接外部服務進 Claude |
| [07](./07-schedule/guide.md) | Schedule | 設定定時自動執行的 Claude 任務 |

## 怎麼開始

1. 安裝 [Claude Code](https://claude.ai/code)
2. 從 `01-claude-md` 開始，讀 `guide.md`，跟著練習做
3. 做完一個再往下一個

每個練習都有一個具體任務，做完就算過。不需要任何前置知識。

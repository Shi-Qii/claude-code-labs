## 里程碑定義

**Level 1 完成：**  
能用 Claude Code 處理日常 bug 修復，習慣在開始前先看 `git diff`，知道 `/clear` 和 ESC 的用途。

**Level 2 完成：**  
有自己的自訂 `/指令`，Claude 認識你的專案，MCP 接了至少一個外部工具（如 GitHub）。

**Level 3（選用）：**  
能自己寫 MCP server 或 Claude API 程式，或把 Claude Code 整合進 CI/CD 流程。

---

## 使用建議

- 每週一次 1:1 或團隊 sync，5 分鐘更新一下進度表
- 完成 Level 1 後，請成員帶一個「用 Claude Code 解決的真實問題」分享給團隊
- 進度落後不是問題，重點是確保每個人的卡點都有人幫解決

# 團隊導入工具包

這個資料夾給**想帶同事學 Claude Code** 的人用。

一般學習者不需要進來，主線課程在 [基礎篇（00-07）](../README.md) 和 [進階練習/](../進階練習/README.md)。

---

## 這裡有什麼

| 檔案 | 用途 | 適合誰 |
|------|------|--------|
| [01-team-setup.md](./01-team-setup.md) | 共用設定教學：讓新同事 clone 即可用 | 負責導入的工程師 |
| [02-workshop.md](./02-workshop.md) | 90 分鐘工作坊腳本 | 要辦內部教育訓練的人 |
| [03-progress.md](./03-progress.md) | 學習進度追蹤範本 | 帶團隊的技術 lead |
| [04-security.md](./04-security.md) | 公司環境注意事項 | 導入前的資安確認 |
| [案例集/](./案例集/) | 4 個實戰走查（Java/Spring Boot） | Demo 素材 / 自學參考 |

---

## 建議順序

```
導入前準備：04 資安確認 → 01 共用設定
辦工作坊：  02 工作坊腳本（案例集作為 demo 素材）
追蹤進度：  03 進度表
```

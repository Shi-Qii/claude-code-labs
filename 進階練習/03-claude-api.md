# 03 · Claude API / Agent SDK

**直接用程式碼呼叫 Claude，打造自己的 AI 工具**

---

## 你會做到什麼

用 `@anthropic-ai/sdk` 寫一個小型 agent，能自動完成多步驟任務（例如：讀資料夾 → 分析程式碼 → 輸出報告）。

---

## 需要先完成

- Node.js 或 Python 基礎
- Anthropic 帳號（有 API key）

---

## 你會學到什麼

- Messages API 基本結構
- Tool use（讓 Claude 呼叫你定義的函式）
- 串接多輪對話
- Streaming 即時輸出
- Token 計算與費用估算
